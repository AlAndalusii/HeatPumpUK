## ðŸ“š Using the SEO Template

### For New Articles, Follow This Process:

1. **Pre-Writing Phase**
   - Select primary keyword (see Action Plan for keyword research)
   - Read: Template in `SEO_QUICK_REFERENCE.md`
   - Fill out pre-publication checklist

2. **Writing Phase**
   - Write naturally (don't force keywords)
   - Follow heading hierarchy: H1 > H2 > H3
   - Include keyword in first sentence naturally
   - Keep content user-focused

3. **Optimization Phase**
   - Create title tag (45-59 characters, keyword-rich)
   - Write meta description (150-160 characters)
   - Set URL slug (clean, keyword-focused)
   - Ensure single H1, keyword present
   - Review for natural language

4. **Pre-Publication Phase**
   - Complete checklist in `SEO_QUICK_REFERENCE.md`
   - Final review for typos and clarity
   - Test on mobile device
   - Publish and promote
